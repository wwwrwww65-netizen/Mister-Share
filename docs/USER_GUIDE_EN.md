## How to Send Files

### Step 1: Select Files
1. Open MisterShare
2. Tap on **Files**, **Photos**, **Videos**, **Music**, or **Apps**
3. Select the files you want to send
4. Tap **Send Selected**

### Step 2: Connect to Receiver
1. Wait for the receiver to appear in the list
2. Tap on their device name
3. Enter the 6-digit PIN shown on the receiver's screen
4. Tap **Connect Securely**

### Step 3: Transfer
- The transfer will start automatically
- You can see real-time speed and progress
- You can pause/resume the transfer anytime
- Wait for completion

---

## How to Receive Files

### Step 1: Start Receiving
1. Open MisterShare
2. Tap **Receive** on the home screen
3. A 6-digit PIN will be displayed

### Step 2: Share PIN
1. Tell the sender your 6-digit PIN
2. They can also copy it by tapping on it
3. Wait for the sender to connect

### Step 3: Receive
- Files will be received automatically
- Check progress on the transfer screen
- Files are saved to your Downloads folder

---

## Security Features

### PIN Authentication
- Every transfer requires a 6-digit PIN
- PINs expire after 30 minutes
- New PIN for each session

### AES-256 Encryption
- All files are encrypted during transfer
- Military-grade security
- End-to-end encryption

### File Verification
- MD5 checksum for every file
- Automatic corruption detection
- Failed transfers are retried

## Troubleshooting

### Connection Failed
**Problem**: Can't connect to another device

**Solutions**:
1. Make sure both devices have WiFi and Location enabled
2. Grant all required permissions
3. Try moving closer to each other
4. Restart both apps

### Transfer Slow
**Problem**: Transfer speed is very slow

**Solutions**:
1. Check if devices support 5GHz WiFi
2. Move devices closer together
3. Close other apps using WiFi
4. Ensure battery saver is off

### PIN Not Working
**Problem**: PIN is rejected

**Solutions**:
1. Make sure you entered the correct PIN
2. Check if PIN has expired (5 minutes)
3. Try creating a new session

### File Not Received
**Problem**: File didn't save properly

**Solutions**:
1. Check available storage space
2. Grant storage permissions
3. Check Downloads folder
4. Try transfer again

## FAQ

### Q: Do I need internet?
**A:** No! MisterShare works completely offline using WiFi Direct.

### Q: How fast is it?
**A:** Up to 80+ Mbps on 5GHz WiFi, around 20 Mbps on 2.4GHz.

### Q: Is it secure?
**A:** Yes! All transfers are encrypted with AES-256 and require PIN authentication.

### Q: What file types are supported?
**A:** All file types - photos, videos, music, documents, APKs, and more.

### Q: Can I transfer multiple files?
**A:** Yes! You can queue multiple files and transfer them all at once.

### Q: Where are received files saved?
**A:** Files are saved to your Downloads folder by default.

### Q: Can I pause a transfer?
**A:** Yes! You can pause and resume transfers anytime.

### Q: Is there a file size limit?
**A:** No technical limit, but large files (>1GB) may take time.
